# GitHub Repository Setup Guide

## Quick Start

Your plugin is ready to push to GitHub! Follow these steps:

### 1. Add All Files to Git

```bash
git add .
```

### 2. Make Your First Commit

```bash
git commit -m "Initial commit: AI Store Assistant WordPress plugin v1.0.0"
```

### 3. Add GitHub Remote

```bash
git remote add origin https://github.com/bedigitalsi/woo-ai-asistent.git
```

### 4. Push to GitHub

```bash
git branch -M main
git push -u origin main
```
